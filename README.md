# videos_analyzer_cli
## Постановка задачи

Требуется написать cli приложение для обработки csv-файлов с метриками видео на YouTube.
Приложение читает файлы с данными о видео и формирует отчеты. Нужно сформировать один отчёт, в котором будут перечислены кликбейтные видео - те, у которых одновременно высокий CTR и низкое удержание.

## Описание решения

Файловый состав решения:
1. main - главный исполняемый файл приложения;
2. run - содержит примеры команд для запуска cli приложения;
3. tests_data - содержит файлы использующиеся для тестов и тестовых запусков приложения;
4. tests - содержит первичные тесты основных компонентов (за исключением processors) приложения;
5. в папках reports_presenters, reports_makers, reports, processors, data_extracters, common содержатся файлы содержащие исходные коды основных компонентов приложения;

Исходная задача была разбита на следующие подзадачи:
1. парсинг аргументов приложения;
2. чтение данных из файлов;
3. анализ данных и формирование отчета;
4. визуализация отчета.

Для решения этих задач были реализованы следующие компоненты:
1. main - содержит код парсинга (argparse) и координации остальных компонентов;
2. data_extracters - компонента описывающая процесс получения данных из файлов или других источников;
3. reports_makers - компонента описывающая логику формирования конкретного типа отчета по соответствующим данным;
4. reports_presenters - компонента описывающая логику отображения данных конкретного типа отчета в консоль;
5. processors - компонента описывающая логику объединения компонент описывающих чтение, анализ и представление данных в виде отчета;
6. common, reports - содержат сущности используемые для представления в коде данных, используемых остальными компонентами.
